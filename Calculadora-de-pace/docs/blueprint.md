# **App Name**: Corre Junto

## Core Features:

- Motor de Corrida Completo: Motor de cálculo abrangente para determinar pace, tempo de chegada, distância total e conversões de velocidade em esteira com atualizações em tempo real.
- Simulador de Estratégia de Prova: Gera gráficos de parciais personalizados para estratégias negativas, constantes ou positivas, fornecendo tempos por quilômetro e orientação de esforço.
- Cofre de Anamnese Digital: Salva e sincroniza fichas de avaliação atlética, incluindo histórico de lesões e níveis de condicionamento, diretamente no banco de dados para revisão profissional.
- Insight de Pace com IA: Analisa os recordes históricos e a constância do usuário como uma ferramenta para decidir quando recomendar paces alvo ou avisos de risco de fadiga.
- Mapeamento de Zonas de FC: Mapeamento fisiológico automático baseado em fórmulas de Karvonen e LTHR usando entradas de FC Máxima, Repouso e Limiar.
- Planejador de Hidratação e Géis: Ferramenta consciente do ambiente que calcula a ingestão precisa de líquidos e carboidratos com base no peso, intensidade e fatores climáticos.
- Login Social com Google: Autenticação segura em um toque via Google Sign-In usando Firebase Auth para acesso unificado à conta.

## Style Guidelines:

- Cor primária: Orquídea Vívida (#a761d4), representando performance de elite. Fundo: Obsidiana Escura (#17121b) para alto contraste.
- Cor de destaque: Verde Ácido (#5cbf34) para botões de ação e status de resultados. Acentos secundários em Ardósia (#3b2d45).
- Fonte para títulos e corpo: 'Inter'. Oferece um visual neutro e legível para dados numéricos densos.
- Interface rica em símbolos usando ícones de linha monocromáticos para estados de corrida, hidratação e métricas biológicas.
- Layout segmentado em cartões e abas, enfatizando a hierarquia de blocos de dados para facilitar o uso em dispositivos móveis.
- Transições dinâmicas entre grupos de calculadoras com micro-interações nos seletores (chips) para indicar estado ativo.
